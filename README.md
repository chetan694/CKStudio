# Chetan Khajuria Music Website

Premium personal music website for Chetan Khajuria, built with React, Vite, HTML, CSS, and JavaScript.

## Setup

Install dependencies:

```bash
npm install
```

Create `.env` from `.env.example` and add your EmailJS values.

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Environment Variables

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
VITE_CONTACT_EMAIL=chetankhajuria77@gmail.com
```
